# Resume_Generator

#### Video Demo: [<URL HERE>](https://www.youtube.com/watch?v=nn_tgibN0NI)

#### Description:
Resume Generator is a Python application that creates a professional one-page resume in PDF format using the FPDF library. The program collects user information such as personal details, education, experience, and skills through the terminal, then automatically generates a clean and organized resume.
    
The project was developed as a final project for CS50P and demonstrates the use of several Python concepts including:
* Object-Oriented Programming (OOP)
* Classes and methods
* File generation with FPDF
* Regular expressions for validation
* Exception handling
* Lists and dictionaries
* Functions and modular programming

#### How to Run
Install dependencies:
```bash
pip install fpdf2
```

Run the project:
```bash
python project.py
```

Example Input
``` bash
First name: Ilyass
Last name: Zaid
Email: ilyass@example.com
Phone: 0612345678
Address: Agadir, Morocco
Profile summary: Motivated software developer passionate about Python and web development.
```
Output

The program generates a file called:
Resume.pdf
containing the formatted resume.
